Getting Started with Corona

X Comments corona learning

I’ve been working on my game for about three months now and I feel like I’m in a pretty good spot. I have a good grasp of LUA and the Corona SDK and my problems are more focused on what type of game to make and not how to make a game. I’ve spent a lot of time reading documentation and taking tutorials and I want to share the resources that helped me get started with any fellow beginners out there.

Coronalabs.com

If you have a question about Corona, then there is a good chance you can find the answer in Corona’s resource documentation. I’ve spent a a lot of time going through their university, API docs, forums, and blog. I also recommend taking a look at the Corona Geek Videos and subscribing to the Corona Labs Digest, which both keep you up to date with Corona’s features and enhancements.

Masteringcoronasdk.com

After poring over Corona’s official website, I took the free beginner’s game development crash course. The free tutorial was done well, so I sprung for the beginning mobile game development video tutorial series. If I could go back in time, then I think I would have started out by taking these video tutorials before reading so much of Corona’s official documentation.

Scene Management

Scene management is a core concept in Corona and develephant.net, a wonderful Corona resource, has a great storyboarding tutorial explaining how it all works. Corona has since moved from the storyboard API to the new composer API, but the tutorial is still good at explaining the concepts of scene management. Corona has since published a tutorial on their composer API, which is what you’ll want to use now.

Classes & Inheritance

My original code got unwieldy very quickly and I’ve refactored my code to use objects and inheritance in an effort to tame the beast. Two of the tutorials that helped me understand how OOP works in Corona are RPG Themed OOP Design Pattern for Corona and Form Zero to OO – ArdentKids guide to OOP with LUA and Corona.

Game Assets

I found most of my game assets at opengameart.org and spritedatabase.net. I use imagesplitter.net to split up the single images into multiple frames and then use TexturePacker to create sprite sheets ready to use with Corona. I recently wrote a couple of tutorials demonstrating how I’ve used the various services for my sprite needs.
