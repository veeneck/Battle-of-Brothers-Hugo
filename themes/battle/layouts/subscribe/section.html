What is an RSS Feed?

RSS allows you to get automatic updates when a new blog post is created. Using an app like Feedly for the web, or Flipboard on a mobile device, you can be notified when we write something new. There are tons of free readers that will help you keep up to date with topics you're interested in, so go and set one up!
